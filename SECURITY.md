# Security Policy

## Reporting a Vulnerability

This repository is a public GitHub profile README and job-board data files. It does not run a production service or store secrets.

If you find a security issue (e.g. malicious content in a pull request, workflow abuse), please report it privately:

- **Email:** Use GitHub's [private vulnerability reporting](https://github.com/nicholasg3/nicholasg3/security/advisories/new) on this repository, or contact Nicholas Garcia via his NUS faculty page.

Please do not open public issues for undisclosed security problems.

## Scope

- In scope: GitHub Actions workflows, `data/jobs.yaml` content integrity, supply-chain in CI dependencies.
- Out of scope: Third-party application links listed in job postings (e.g. Google Docs JDs).

## Supported versions

Only the `main` branch is maintained.
